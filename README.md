# Django | Class-Based-CRUD
## RECUIREMENTS:

    Python 3.6
    Django 2.1

## To RUN The Project:
    Clone the git
    Open the project
    Open terminal and write two commands(for database purposes) 
      python manage.py makemigrations 
      python manage.py migrate
    Run the project using the command 
      python manage.py runserver
